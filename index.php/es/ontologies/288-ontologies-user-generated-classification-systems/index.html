Ontologies from User-Generated Classification Systems

Esta ontologia fue desarrollada como parte de la tesis doctoral “A Method for Developing Ontologies from User-Generated Classification Systems”. La ontologia se diseñó para proveer descripciones semánticas de los datos de entrada y de salida de los diferentes procesos que componen el método que se propone en esta tesis. Junto con la ontología se publica un archivo readme.pdf donde se puede encontrar información adicional.
